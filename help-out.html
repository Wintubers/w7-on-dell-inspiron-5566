Help out with the project

Do you want to help out with this project? Well you can, just follow these steps.


Step 1: Choose what you want to help about

The issues page is only going to be used if the mod doesn't work or if the website is broken.


Step 2: Fix it

If it's an issue with documentation, it will be easy to fix, if it's a problem with the mod... Well then looks like this could take you a while, if you do fix an issue with the ISO, continue to step 3.


Step 3: Make a Pull Request

Make a Pull Request to the Github repo and wait for us to review it, if we think it's a good idea to add it, we will and you will be listed in the helpers section.
