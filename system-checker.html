Windows 7 on the Dell Inspiron 5566

An entire website for getting Windows 7 on the Dell Inspiron 5566

On this page you can check if your device can do mod.

Some models are not supported, so this site will tell you if yours is.

Supported Systems

These will show your CPU, so you should check that (you can in System Viewer or Task Manager)

Intel Core i3-7100U (this is what this is tested on)

Intel Core i5-7200U

Intel Core i7-7500U

Intel Core i5-7300U

Intel Core i7-7600U


If your Dell Inspiron 5566 has one of these CPUs, it will be able to run Windows 7

This might also work on the Dell Inspiron 5567 (not tested, try at your own risk)

Un-Supported Systems

The Dell Inspiron 5566's with this CPU, will NOT WORK

Intel Celeron N3060
